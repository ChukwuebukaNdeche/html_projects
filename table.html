HTML Tag and thier Funtionalities
Tag Name Related Attribute(if any) Funtionality
p tag - Offers a way to structure your text into different paragraph
br - Use to break a sentence, making it start from the next line.
Center content - Center tags are used to put any content in the center of the page.
hr - They are used to visually break-up sections of a document by creatind a horizontal line
Preserve Formatting pre - Use to preserve the formatting of the source document
div - Used as a container for HTML elements
span - This is an inline container used to mark up a part of a text or part of a document
section - Used to group together related element
nav - Uesd to create navigation bars in HTML
label for Used to associate a text label with a form input field. it is used to tell users the value that should be filled.
input type, value, name Used to declare input controls that allows users to input data.
form action Used to create an interractive form on a webpage.
main - Specifies the main content of a document
bold tag or b tag - Anything that appears in the b tag is displayed in bold
italic tag or i tag - Used to italicize a text
u tag - Used to underline a text
strike tag - It displays a text as strikethrough, which is a thin line through the text.
tt tag - The content of tt tags are written in monospaced font.
sup tag - Used to transform a text to superscript.
sub tag - Used to transform a text to subscript.
ins tag - The text in the ins tag are displayed as inserted text.
del tag - The text in the del tag are displayed as deleted text.
big tag - Used to make the font size of a text larger than regular text.
small tag - Used to make the font size of a text smaller than regular text.
mark tag - Used to display a text as marked with yellow ink.
strong tag - Used to display a text as important text.
abbr tag - Used to show the full description of an abbreviation
acronym - Used to indicate that a text is an acronym.
bdo tag or bi-directional override - Used to override the current text direction.
dfn tag - Allows you to specify that you are introducing a special term.
q tag - Used to add a double quote to a sentence
address tag - Used to contain an address.
